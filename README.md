# GeoNORM
**Overview**

This repository contains three numerical case studies used in the paper, organized into the following folders:

- DarcyCase
- PipeCase
- BloodCase

Each folder corresponds to an independent experiment presented in the paper.

**Folder Structure**

Within each case folder, the following components are included:

- Implementation of the proposed method
- Implementations of comparative (baseline) methods


**Environment**

All codes are implemented in Python. 

Please install the required packages before running the code.

**Usage**

Run the corresponding script:

python main.py

**Dataset**

The datasets used in this project are available at:

https://drive.google.com/drive/folders/1d3vY-MYiDJUz6k_TDwbQdp_V7yNcT8zX?usp=drive_link

**Notes**
- The three cases are independent and can be run separately.
- Parameters and model configurations can be modified within each case folder as needed.
